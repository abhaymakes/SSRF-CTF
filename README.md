# Application Setup and Deployment

This repository contains a Python Flask application that runs two servers on different ports. The following guide explains how to set up the environment, install dependencies, and run the servers.

## Prerequisites

Before running the application, make sure you have the following installed:

- **Python 3.x**: You need Python installed on your system.

## Steps to Run the Application

Follow these steps to get your environment ready and start the application.

### 1. Clone the Repository
First, clone this repository to your local machine:

```bash
git clone https://github.com/abhaymakes/SSRF-CTF.git
cd SSRF-CTF
```

### 2. Run the script
Add execute permission to the 'start_lab.sh' file using the following command:
```bash
chmod +x start_lab.sh
```

Finally, run the script:
```bash
./start_lab.sh
```
